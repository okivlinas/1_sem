Основные компоненты

Основным компонентом адаптивного дизайна являются гибкие сетки, которые являются частью резинового дизайна. Применение гибких сеток заключается в установке процентного значения ширины вместо абсолютного в пикселах. Различные ширины экрана, которые указываются с помощью медиазапросов называются точками останова. Стандартными принято считать следующие разрешения: для мобильных устройств 480px, для планшетных компьютеров 768 px, для нетбуков 1024 px, для ПК 1280 px и больше
